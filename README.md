# CSC372 Assignment 7 – Rock Paper Scissors (React)

A Rock Paper Scissors game built with React and Vite.

## Setup

```bash
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.

## How to Play

1. Click **Rock**, **Paper**, or **Scissors** to make a throw.
2. The computer's side will shuffle through choices and make a pick.
3. The result is displayed below the board.
4. The scoreboard tracks wins, ties, and losses across rounds.
5. Click **Reset Game** to start over and clear the score.
